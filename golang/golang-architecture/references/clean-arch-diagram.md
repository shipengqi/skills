# Clean Architecture Diagram

## Dependency Direction

```
┌──────────────────────────────────────────────┐
│  cmd/<app>/                                  │
│  ├── main.go          → app.NewApp().Run()   │
│  └── app/             → InitializeWebServer  │
└───────────────────┬──────────────────────────┘
                    │ calls
                    ▼
┌──────────────────────────────────────────────┐
│  internal/<app>/                             │
│                                              │
│  handler/http/        表现层                  │
│  ├── handler.go       Handler struct         │
│  └── user.go          core.HandleJSONRequest │
│           │ depends on IBiz (interface)      │
│           ▼                                  │
│  biz/v1/<domain>/     业务层                  │
│  ├── biz.go           IBiz aggregate         │
│  └── user/user.go     UserBiz implementation │
│           │ depends on IStore (interface)    │
│           ▼                                  │
│  store/               数据层                  │
│  ├── store.go         IStore aggregate       │
│  ├── user.go          UserStore impl         │
│  └── mock_store.go    mockgen generated      │
│           │ uses                             │
│           ▼                                  │
│  model/               实体层                  │
│  └── user.gen.go      GORM model, zero deps  │
│                                              │
└──────────────────────────────────────────────┘
```

## What Each Layer May Import

| Layer | May import | Must NOT import |
|-------|-----------|-----------------|
| `handler` | `biz.IBiz`, `validation`, `errno`, `contextx`, `core` | `store`, `model` directly |
| `biz` | `store.IStore`, `model`, `errno`, `log`, `contextx` | `handler` |
| `store` | `model`, `errno`, `gorm.io/gorm` | `biz`, `handler` |
| `model` | stdlib, `gorm.io/gorm` tags | anything internal |
| `internal/pkg/` | stdlib, external libs | `internal/<app>/` |

## Layer Responsibilities

### handler — Parse → Validate → Call → Respond

```go
func (h *Handler) CreateUser(c *gin.Context) {
    core.HandleJSONRequest(c, h.biz.UserV1().Create, h.val.ValidateCreateUserRequest)
}
```

Handler never:
- Reads from DB
- Signs tokens
- Applies business rules
- Formats error messages

### biz — Business Rules + Orchestration

```go
func (b *userBiz) Create(ctx context.Context, rq *apiv1.CreateUserRequest) (*apiv1.CreateUserResponse, error) {
    userM := &model.UserM{Username: rq.GetUsername()}
    if err := b.store.User().Create(ctx, userM); err != nil {
        return nil, err
    }
    if _, err := b.authz.AddGroupingPolicy(userM.UserID, known.RoleUser); err != nil {
        log.W(ctx).Errorw("Failed to add grouping policy", "user", userM.UserID)
        return nil, errno.ErrAddRole.WithMessage(err.Error())
    }
    return &apiv1.CreateUserResponse{UserID: userM.UserID}, nil
}
```

Biz:
- Orchestrates multiple store calls
- Applies domain rules (password validation, role assignment)
- Logs internal failures; returns errno sentinels
- Never constructs `*gorm.DB` queries

### store — Data Access Only

```go
func (s *userStore) Create(ctx context.Context, obj *model.UserM) error {
    return s.db.WithContext(ctx).Create(obj).Error
}

func (s *userStore) Get(ctx context.Context, userID string) (*model.UserM, error) {
    var obj model.UserM
    if err := s.db.Where("user_id = ?", userID).First(&obj).Error; err != nil {
        if errors.Is(err, gorm.ErrRecordNotFound) {
            return nil, errno.ErrUserNotFound
        }
        return nil, errno.ErrDBRead.WithMessage(err.Error())
    }
    return &obj, nil
}
```

### model — Data Structs Only

```go
// model/user.gen.go — generated by gorm gen, never hand-edited
type UserM struct {
    gorm.Model
    UserID   string `gorm:"column:user_id;uniqueIndex"`
    Username string `gorm:"column:username;uniqueIndex"`
    Password string `gorm:"column:password"`
}

func (*UserM) TableName() string { return "user" }
```

## Transaction Pattern

Transactions span biz-layer logic via `store.TX()`:

```go
func (b *userBiz) Transfer(ctx context.Context, ...) error {
    return b.store.TX(ctx, func(ctx context.Context) error {
        if err := b.store.User().Debit(ctx, fromID, amount); err != nil {
            return err
        }
        return b.store.User().Credit(ctx, toID, amount)
    })
}
```

The tx is injected into ctx inside `store.TX`; sub-calls pick it up via `store.DB(ctx)`.
